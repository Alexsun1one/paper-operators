# State Coding / 状态编码

State coding carries exact status — degree, stage, outcome, quality — with no extra words. Where `references/primitives.md` supplies the nouns of a scene, state coding supplies its precision: how each primitive is drawn so its status reads instantly and consistently. Consistent encodings are what keep a complex multi-state scene readable instead of noisy, and reusing the same encoding for the same state across a series is what lets a reader trust the pictures.

Read this with `references/style-dna.md` for the palette and label rules, and with `references/primitives.md` for the objects these encodings attach to.

## Color Semantics / 色彩语义

Formalize the palette named in `style-dna.md`. Each color means one thing; reuse it that way across the set.

- **Ink / Charcoal · 墨色** — structure and outline; the bones of the scene.
- **Paper Off-White · 米白** — neutral body / default state; the unremarked baseline.
- **Quiet Gray · 静灰** — inactive, background, or done; what is present but not live.
- **Passage Blue · 流蓝** — the live route, current flow, active reader path.
- **Evidence Amber · 证珀** — proof, attention, the thing to notice.
- **Risk Coral · 风险珊瑚红** — failure, cost, stop, the danger.
- **Growth Mint · 生长薄荷绿** — gain, healthy, improved, the positive change.
- **Focus Accent · 焦点色** — the single most important point in the image; chosen per scene.

Rule: keep ≤ ~4 semantic colors active in one image. Use exactly one focus accent — never two points competing for the eye. Always pair color with form (thickness, texture, posture); color reinforces status, it should never be the only thing carrying it.

## Path Encoding / 路径编码

How a carrier ribbon or route strip reads at a glance.

- **Thickness** — importance or volume; a thicker ribbon carries more weight or traffic.
- **Texture** — solid = certain / confirmed; dashed = tentative / proposed; dotted = broken or intermittent.
- **Color** — status, per Color Semantics above (passage blue live, risk coral failing, quiet gray done).
- **Direction markers** — small folded chevrons or arrow tabs show flow direction.
- **Width taper** — a ribbon that narrows along its length shows flow or volume reducing.
- **Broken vs mended** — a snapped ribbon with a stitched or patched join shows a failure that has been repaired.

## Elevation & Layering / 高度与层次

Depth carries priority without words.

- **Raised** — active, current, or priority; the object is lifted off the tabletop.
- **Recessed** — background, inactive, or done; the object sits lower and quieter.
- **Stacked** — hierarchy; higher layers depend on or build from lower ones.
- **Overlap** — a paper object lying over another reads as on top of, newer, or masking it.

## Gate / Threshold States / 闸门与阈值状态

A gate's posture is its verdict.

- **Open / 开** — pass; the flow goes through.
- **Ajar / 半开** — hold; conditional, partial, or pending.
- **Closed / 关** — blocked or not yet reached.
- **Locked / 锁** — reject; hard-stopped, sealed.
- Pair with a **stamp** (通过 / 搁置 / 拒绝) only when the posture alone could read as ambiguous.

## Edge & Surface States / 边缘与表面状态

The condition of a paper object tells its history.

- **Crisp / 平整** — new, clean, untouched.
- **Torn / 撕裂** — damaged, broken, or contested.
- **Mended / 缝合** — torn then repaired; recovered.
- **Sealed / 封口** — closed off, protected, or final.
- **Dog-eared / 折角** — used, marked, or revisited.
- **Creased / 折痕** — worn from repeated handling; well-used, not damaged.
- Surface: **clean / 干净** vs **marked / 标记** vs **stamped / 盖印** — unremarked, annotated, or officially decided.

## Light & Emphasis States / 光与强调状态

Where the light is, the eye goes.

- **Lit / 明亮** — present and legible.
- **Dim / 暗淡** — present but secondary; recessed importance.
- **Spotlight / 聚光** — the focus accent; the one point pulled forward.
- **Shadowed / 投影** — cost, consequence, or footprint cast behind something.
- **Backlit / 透光** — a translucent layer lit from behind to reveal hidden structure beneath it.

## Tab & Marker States / 标签与标记状态

How a tag, tab, or marker encodes membership and status.

- **Filled vs outline** — filled = active / complete; outline = pending / empty.
- **Checked vs flagged** — a check mark = done or passed; a flag = at-risk or to-review.
- **Grouped by color** — tags sharing a color belong to one path, state, or category.
- **Sequence** — tag order, row position, or a number shows where something falls in a series.

## Degree & Quantity / 程度与数量

Show amount and intensity with form, not numbers.

- **Size** — bigger means more, or more important.
- **Count** — the number of cards, tags, or wells shows quantity.
- **Slider position** — where the marker sits along a track shows the level.
- **Scale tilt** — a balance beam's lean shows which side outweighs.
- **Stack height** — a taller stack means more accumulated.
- **Gauge needle** — the needle's angle shows intensity or fill.
- **Fill level** — how full a well, bin, or jar is drawn shows quantity remaining or consumed.

## Operator Expression States / 操作员表情状态

The operator's face is a state encoding too. A few hand-drawn line strokes read the operator's relationship to the work — how hard, how sure, how settled — without any text. Use it only when that state matters; otherwise keep the face plain.

- **Neutral / 平静** — two straight brow/eye lines, a flat mouth; the default, when no state needs reading.
- **Effort / 吃力·专注** — brows angled down and inward, mouth a tight short line; the operator is straining or concentrating (e.g. holding a heavy tradeoff, pulling a taut ribbon).
- **Focus / 凝神** — narrowed brow lines turned toward the work, mouth small and flat; careful inspection.
- **Strain / 受压** — brows pinched, mouth a small downward line; the work is heavy or going wrong (pairs with risk coral nearby).
- **Relief / 舒展** — brows lifted and relaxed, mouth a small upward curve; a beat has resolved (pairs with passage blue or growth mint).
- **Satisfaction / 落定** — eased brows, a calm upward mouth; the deliverable is done (e.g. after the stamp lands).
- **Care / 照料** — soft, lowered brows turned toward a fragile object; tenderness, in life/psychology scenes.

Rules: one expression per operator; abstract line strokes only, no realistic features; profile or three-quarter, looking at the work, never head-on; in a series, progress the expression with the throughline (e.g. focus → strain → relief). A single red line may mark a stronger mouth beat. If removing the face changes nothing about the meaning, keep it neutral.

## Encoding Budget / 编码预算

Do not stack encodings until the scene screams.

- Pick the 1–2 encodings that carry the article's precision (e.g. path texture + one color, or elevation + a gauge).
- Everything else stays neutral — off-white, quiet gray, crisp edges, flat lighting.
- If a scene needs a third encoding, ask whether the article truly requires it or whether the image is becoming a dashboard.
- One focus accent, always; the rest of the color budget is shared by the active encodings only.

## Accept / Regenerate

Accept:

- Status reads in three seconds with no extra text.
- Each color, texture, or posture means one consistent thing across the set.
- The scene uses ≤ ~4 semantic colors and exactly one focus accent.
- Only the encodings that carry the article's point are active; the rest stay neutral.
- Broken, torn, or dim states are used to mean something, never as decoration.

Regenerate:

- Two points fight for the focus accent.
- Too many simultaneous encodings make the scene read as a dashboard.
- A color or texture is used decoratively with no assigned meaning.
- Status relies on unreadable tiny text instead of form, color, or posture.
- Encodings contradict each other (e.g. a "done" object lit brighter than the live one).
