# Primitives / 原元素

The reusable atomic paper parts every Paper Operators scene is built from. Primitives are the nouns of the kit; the operator's action is the verb. Pick primitives first, then arrange them. Reuse the same primitive for the same meaning across every image in a series — that consistency is what makes a multi-image set feel like one world instead of ad-hoc objects.

Read this with `references/style-dna.md` (palette, label rules) and `references/operator-library.md` (the verbs and tools).

## Paper-Object Primitives / 纸物原元素

Each item: concept it carries — material/fold note — labels it commonly hosts.

### Carriers & Paths / 载体与路径

- **Ribbon / 绸带** — route, relationship, or reader path; pleated or folded paper strip; hosts path labels, distance ticks, stage markers.
- **Route Strip / 路径条** — a fixed linear sequence; flat folded band; hosts station tabs and step numbers.
- **Rail / 滑轨** — a track something moves or slides along; folded U-channel; hosts position ticks.
- **Thread / 牵线** — a tenuous connection, dependency, or tension; thin paper line, sometimes knotted; hosts small tie labels.
- **Bridge / 桥** — a handoff or transition across a gap; folded span between two points; hosts interface labels.
- **Knot / 结** — a binding, dependency, or entanglement; folded interlock; hosts relation labels.
- **Junction / 接口** — where paths meet, split, or merge; folded intersection node; hosts flow-direction markers.

### Surfaces & Holders / 台面与托件

- **Card / 卡片** — one discrete unit, idea, or state; flat folded tile; hosts a single short label.
- **Tray / 托盘** — a collected set that travels together; shallow folded basin; hosts a tray label and group tags.
- **Board / 板** — a working or staging surface; flat rigid panel; hosts a board title, pins, or tabs.
- **Table / 操作台** — the scene of action itself; the tabletop plane; hosts station labels.
- **Shelf / 层架** — ranked or archived items; stacked folded ledges; hosts row or level tags.
- **Plinth / 底座** — one object elevated for importance; folded pedestal; hosts a caption plaque.
- **Slab / 层板** — a single layer in a stack; flat folded plate; hosts a layer name and number.
- **Mat / 衬垫** — quiet neutral ground or framing edge; flat thin pad; hosts edge labels.

### Enclosures & Boundaries / 围合与边界

- **Room / 房间** — a bounded context or domain; folded walls; hosts a room name.
- **Screen / 屏风** — a soft separator that can be seen around; folded standing panel; hosts surface labels.
- **Gate / 闸门** — a pass, filter, or permission threshold; hinged folded panel; hosts gate-state labels.
- **Window / 窗口** — a view in, or a limited access point; cut opening in folded paper; hosts a caption.
- **Frame / 画框** — what is selected, curated, or interpreted; folded border; hosts a frame label.
- **Envelope / 信封** — something sealed, held, or awaiting opening; folded flap; hosts a status tag.
- **Box / 盒** — containment, quarantine, or storage; folded carton; hosts a bin label.
- **Drawer / 抽屉** — archived or retrieved content; sliding folded tray; hosts index tabs.
- **Soft Boundary / 软边界** — a limit that is felt, not hard; a low paper strip or traced arc; hosts a quiet edge label.

### Optics & Light / 光学与光

- **Lens / 镜片** — attention, inspection, magnification; translucent folded disc; hosts a target label.
- **Light Card / 光卡** — a directed pool of attention; translucent panel held up; hosts the thing it lights.
- **Lamp / 灯** — a source of emphasis or atmosphere; folded paper lamp; hosts mood tags.
- **Reflector / 反光板** — redirecting attention; translucent angled panel; hosts direction markers.
- **Shadow Strip / 影条** — cost, footprint, or consequence cast behind something; flat shadow band; hosts an effect label.
- **Viewing Window / 取景窗** — the chosen frame of perception; cut opening; hosts a caption.

### Measures & Tools / 度量与工具

- **Scale / 天平** — tradeoff, balance, or weight of evidence; folded balance beam; hosts side labels and counterweight tags.
- **Slider / 滑块** — degree, level, or adjustable value; folded marker on a track; hosts min/max ticks.
- **Gauge / 仪表** — a reading of intensity or status; folded dial; hosts a needle position and unit label.
- **Ruler / 尺** — a standard of comparison or measure; flat folded rule; hosts unit ticks.
- **Stamp / 印章** — approval, decision, or mark of record; folded handle and pad; hosts a verdict word.
- **Brush / 笔刷** — tune, refine, restore, or clean; folded brush head; hosts the surface it works on.
- **Sieve / 筛** — filtering, selection, or separation; folded mesh panel; hosts pass/retain labels.
- **Valve / 阀门** — regulating flow, opening or closing a stream; folded rotating piece; hosts open/closed tags.
- **Tuning Fork / 音叉** — calibration, harmony, or fine adjustment; folded fork; hosts a target-frequency label.

### Containers of State / 状态容器

- **Folder / 文件夹** — a grouped state or case awaiting action; folded jacket with a tab; hosts a case label and status tag.
- **Archive Drawer / 归档抽屉** — stored, lineage, or historical state; sliding folded tray; hosts index tabs.
- **Tag Rack / 吊牌架** — a set of comparable states or attributes; folded rack of hang tags; hosts grouped tag labels.
- **Well / 凹槽** — a held or measured quantity; folded depression; hosts a fill-level label.
- **Bin / 收集箱** — collected residue or output; shallow folded bin; hosts a contents label.
- **Jar / 罐** — accumulated, preserved, or scarce quantity; folded jar form; hosts a fill line and a contents label.

## Label-Container Primitives / 标签容器原元素

Labels live on objects, never in empty space. Mirror the label rules in `style-dna.md`: short readable Chinese labels by default (2-6 characters), attached to objects.

- **Tab / 纸签** — a small folded tab on an edge; use for a short name or state on a card, slab, or drawer.
- **Plaque / 铭牌** — a flat caption card; use to title a plinth, frame, or room.
- **Hang Tag / 吊牌** — a tag hanging from an object by a paper loop; use for price-, status-, or attribute-style labels.
- **Stitched Card / 缝合卡** — a card "stitched" onto a surface; use for labels that must feel attached and permanent.
- **Route Sign / 路牌** — a small directional sign on a path; use for stage, direction, or station labels.
- **Gallery Caption / 展签** — an art-museum caption beside an object; use in art, culture, or curation scenes.
- **Sentence Strip / 句条** — one readable short sentence; use only when it sharpens the takeaway.
- **Stamp / 印记** — an inked verdict on a surface; use for approval, rejection, or decision status.

## The Paper Person Kit / 纸片人构造件

The operator is assembled from the same kinds of paper parts as the scene. The tool does the storytelling; the body stays quiet.

- **Head tiles** — a folded tile, hood, or wedge as the head; calm and plain by default.
- **Expression** — optional: a few hand-drawn line strokes (two for brows/eyes, one for the mouth; profile or three-quarter, looking at the work) that encode the operator's state — effort, focus, strain, relief, satisfaction, care. Keep it abstract and sparse to avoid the uncanny valley; never realistic, never head-on, never Xiaohei white-dot eyes. See `references/state-coding.md` and `references/style-dna.md`.
- **Torso shapes** — rounded rectangle, capsule, tile, wedge, arc, or strip stack.
- **Limb rules** — only the limbs needed to perform the action; quiet otherwise.
- **Tool mounts** — the hand or body presents exactly one domain-specific tool.
- **Color-tab placement** — one small semantic color tab on the operator or tool, no more.
- **Scale** — the operator is 8-18% of canvas height; secondary to the idea, essential to the relation.

Reaffirm: no hair, no costume identity, no realistic facial features. The face is either plain or one simple state-coding expression. Not Xiaohei — no black blob body, no white-dot eyes, no tiny thin legs, no doodle mascot. Not a mascot — not cute-first, not grinning at the reader, not waving, presenting, a sticker, a logo, a robot, an avatar, or a silhouette.

## Carrier-Ribbon Variants / 载体绸带变体

The blue carrier ribbon is the reader path. Its form implies what the flow is doing.

- **Straight / 直行** — a simple uninterrupted sequence from start to end.
- **Branching / 分叉** — a decision point where the path splits into alternatives.
- **Looping / 环回** — a feedback loop, repetition, or return to an earlier step.
- **Broken / 断裂** — a failure, drop, or disconnection in the flow.
- **Merging / 汇合** — separate streams joining into one.
- **Climbing / 攀升** — effortful progression or rising value over stages.

## Composition Primitives / 构图原元素

- **Focal module** — the single object or action the eye lands on; only one per image.
- **Quiet zone** — generous margin and breathing space around the focal module.
- **Depth layers** — foreground props, mid action, back shelves; build a tabletop diorama, not a flat poster.
- **High-angle staging** — shallow isometric tabletop view; raise active objects, recess done/background ones.

## Assembly Rule / 装配规则

- Choose the simplest sufficient set of primitives; add more only when the article's precision needs them.
- One primitive carries one meaning; do not overload a single object with two unrelated jobs.
- Assign each meaning to its primitive once, then reuse the same primitive for that meaning across the series — this is the foundation of the 串联 layer in `references/series-and-chaining.md`.
- Keep semantic colors and label containers consistent with `references/state-coding.md` and `style-dna.md`.

## Accept / Regenerate

Accept:

- Every object is a named primitive from this kit.
- The same meaning maps to the same primitive across the image set.
- Labels sit on label-container primitives, never in empty space.
- The paper person is built only from the Paper Person Kit.
- The primitive set is the simplest that carries the idea.

Regenerate:

- An off-kit ad-hoc object appears that no primitive accounts for.
- A stock icon, clipart, or PPT shape stands in for a primitive.
- The operator gains a face, eyes, mouth, or mascot identity.
- One primitive is forced to mean two unrelated things.
- The scene is a flat poster with no depth layers or focal module.
