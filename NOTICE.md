# Notice

Paper Operators / 纸片人 is an AI-agent skill for generating article illustrations with faceless folded-paper action characters.

The visual system is intentionally distinct from Ian Xiaohei Illustrations and other public illustration IPs. It learns from the general workflow idea that a recurring figure should perform the core conceptual action, but it does not copy Xiaohei's black blob body, white-dot eyes, thin legs, doodle identity, character shape, or example compositions.

When adapting this repository, keep the skill name or provide attribution to Sun Wuyuan in derived documentation.
